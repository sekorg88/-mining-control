// QUICK START

Up and running
in 5 minutes

1
DOWNLOAD MINING CONTROL

Download MiningControl.exe from GitHub Releases. No installation needed — just run the exe.

2
DOWNLOAD MINERS

Download the miners you need: T-Rex (ZANO/NEOX/KRGN), lolMiner (NEXA/IRON), GMiner (BEAM), TT-Miner (EPIC/EVR).

3
RUN AND CONFIGURE

Run MiningControl.exe, complete the setup wizard: enter your wallet addresses and point to your miner executables.

4
RUN BENCHMARK

Let the program measure your real hashrate for each coin (~2 min each). Or enter manually if you already know it.

5
ENABLE AUTO-SWITCH AND MINE!

Turn on Auto-Switch and let the program do the rest. It will automatically mine the most profitable coin 24/7.
